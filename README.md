# 🧮 Math Race Bot

🚀 Telegram math game with levels, XP system, achievements and leaderboard.

---

## 🎮 Features

- 🎯 Difficulty levels (Easy / Medium / Hard)
- 🔢 Choose number of questions (5 / 7 / 10 / 15)
- ⚡ Timer-based bonus system
- 🏆 Leaderboard (Top players)
- ⭐ XP & Level system
- 🏅 Achievements
- 🔁 Play again button
- 📊 Monthly user statistics
